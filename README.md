## Bem-vindo(a) ao Face UMF frontend

Antes de tudo, clone o projeto:

```
git clone https://github.com/eualex/data-faces-frontend.git
```

entre na pasta do projeto

```
cd data-faces-frontend
```

agora, no seu terminal, execute
